## About nerd font

Nerd font is disabled by default. If you want to enable nerd font display:

1. Using a nerd font;
   ```
   $ brew install mplus-nerd-font
   $ # Use this font in terminal
   ```
2. Set a environment variable;
   ```
   export NERD_FONT=true
   ```
